# 📸 Evidencias — nella-frontend

Capturas, logs y documentación de cambios visuales y funcionales del módulo frontend.

## Convención de nombre de archivos

```
YYYYMMDD-[tipo]-[descripción-corta].[ext]
```

Ejemplos:
- `20260218-feat-auth-login-ui.png`
- `20260218-fix-proxy-middleware-log.txt`
- `20260218-refactor-auth-colors.png`
